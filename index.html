Press Shift+Space to toggle
Standard 2-Bulsik (Dubeolsik) Layout:
This is the authentic Korean keyboard layout used in Korea.

Consonants (left hand):
q(ㅂ) w(ㅈ) e(ㄷ) r(ㄱ) t(ㅅ) / a(ㅁ) s(ㄴ) d(ㅇ) f(ㄹ) g(ㅎ) / z(ㅋ) x(ㅌ) c(ㅊ) v(ㅍ)

Vowels (right hand):
y(ㅛ) u(ㅕ) i(ㅑ) o(ㅐ) p(ㅔ) / h(ㅗ) j(ㅓ) k(ㅏ) l(ㅣ) / b(ㅠ) n(ㅜ) m(ㅡ)

Double consonants (Shift):
Q(ㅃ) W(ㅉ) E(ㄸ) R(ㄲ) T(ㅆ)

Examples:
• gksrmf → 한글 (Korean)
• dkssud → 안녕 (hello)
• Rkr → 까 (double ㄱ + ㅏ)

• Backspace decomposes syllables step-by-step (한 → 하 → ㅎ)
• Compound vowels form automatically (ㅗ+ㅏ → ㅘ, ㅜ+ㅓ → ㅝ)
