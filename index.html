3벌식 (Sebeolsik) - Three-Set Layout

About 3-Bulsik: Unlike 2-Bulsik where consonants serve as both initial and final, 3-Bulsik has separate keys for initial consonants (초성, right side), vowels (중성, middle), and final consonants (종성, left side). This allows typing in any order: you can type the final before the vowel!
Note: Many final consonants require Shift (shown as subscript on keys).

Hangul Processing Tools

Decompose (분해)

Breaks down Hangul syllables into their jamo components (초성, 중성, 종성).

Compose (조합)

Combines jamo components into a complete Hangul syllable.

Character Properties (속성)

Decompose String (문자열 분해)

Processes entire strings, decomposing Hangul while preserving other characters.
